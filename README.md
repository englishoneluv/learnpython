# How to use:

### 1. Choose a task(for example, task_sum)
### 2. Go to task folder(task_sum)
### 3. Complete the code in sum.py
### 4. Run tests! Bash command: "pytest -v" in the task directory(task_sum)
```bash
pytest -v
```
### 5. For (4) you need to have Pytest installed on your computer!
### Pytest installation tips:
### a) Ubuntu
```bash
sudo apt install python3-pytest
```
### b) Windows
```bash
pip install pytest
```
### or
```bash
pip3 install pytest
```
