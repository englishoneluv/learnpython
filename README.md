# How to use:

### 1. Clone this repo to your computer
```bash
git clone https://github.com/englishoneluv/learnpython.git
```
### 2. Choose a task(for example, task_sum), go to the task folder(task_sum)
### 3. Complete the code! (In sum.py fix the Sum(a:int, b:int) function)
### 4. Run tests! Bash command: "pytest -v" in the task directory(task_sum)
```bash
pytest -v
```
### 5. For (4) you need to have Pytest installed on your computer!
### Pytest installation tips:
### a) Ubuntu
```bash
sudo apt install python3-pytest
```
### b) Windows
```bash
pip install pytest
```
### or
```bash
pip3 install pytest
```
